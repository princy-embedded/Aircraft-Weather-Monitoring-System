# Aircraft-Weather-Monitoring-System
# Weather Monitoring for Aircraft System and Self-Landing System

## Project Description

This project is an embedded-based aircraft weather monitoring and self-landing assistance system developed using ARM7 LPC2148 microcontroller.

The system monitors environmental conditions such as temperature and weather changes through sensors and supports automated landing assistance during critical conditions.

## Objectives

- Monitor weather conditions in aircraft environment
- Improve aircraft landing safety
- Provide automatic assistance during landing
- Use embedded systems for real-time monitoring

## Technologies Used

- Embedded C
- ARM7 LPC2148
- Keil µVision
- Flash Magic
- UART Communication
- Sensor Interfacing

## Embedded Protocols Used

- UART
- SPI
- I2C

## Components Used

- LPC2148 Development Board
- Temperature Sensor
- Weather Monitoring Sensors
- LCD Display
- Communication Interfaces

## Features

- Real-time weather monitoring
- Sensor-based data acquisition
- Embedded control system
- Self-landing assistance logic
- Serial communication support

## Applications

- Aerospace systems
- Aircraft safety systems
- Embedded automation
- Real-time monitoring systems
